
# Bash Common commands

## Navigation & files
- pwd (Print working directory) -> it shows the current directory
- ls (list) -> list files 
- cd (Change Direcory) Changing the directory 
- mkdir (make directory) 
- touch -> create an empty file
- cp -> copy
- mv -> Moves
- rm -> Remove a file (`rm -r /foldername` to remove a folder -r recursivly )
- cat -> Print out the contents of a file
- tail -> print out the last part of a file, the tail
- tail -f -> prints out the last parts and add new lines that are being printed.
- head -> Show the first lines 

## Search and filter
- grep -> search a pattern
- find -> search files and folders
- wc (wourd count) -> count rows, words bytes
- sort -> sorting

## System and Processes
- man (manual) -> `man ls` show the manual for the ls comand
- echo -> prints text and variables
- env -> shows environmental variables
- chmod (change mode) -> change permissions
- chown (change owner) -> change who owns files and folders
- ps (process status) -> list processes 
- kill -> stops a process 
- `df` / `du` (disk free, Disk Usage)  

## Network
- curl -> Making http calls
- ssh (Secure Shell) -> Logging in to servers 
- scp (Secure Copy Protocol) -> copy files between computers
